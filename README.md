# dev-microservice
DevContainer enabled microservice 

# DevContainer JSON reference:
https://containers.dev/implementors/json_reference/

# Available Features:
https://containers.dev/features

# Initialized SpringBoot with:
spring init --java-version=21 --package-name=com.geek.dev --group=com.geek.dev --artifact=hello-api-aggregation --type=maven-project microsvc 

cd into the java projects dir, microsvc
Create a Dockerfile
Delete the .gitignore and .gitattributes
mvn clean install
